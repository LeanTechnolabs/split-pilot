# Security

This repository contains public product documentation and approved media assets for Split Pilot. It does not contain the production application’s source code.

## Reporting a concern

Do **not** publish vulnerability details, private trip links, or private trip data in public GitHub issues or pull requests.

Please report security concerns privately through the **Feedback** link in the footer on [https://splitpilot.app](https://splitpilot.app). Mark your message as a security report.

When contacting us, include enough information to understand and reproduce the issue. Do not include real user data, real trip information, or live credentials.

Helpful reports typically include:

- A clear description of the issue
- Steps to reproduce using fictional or sanitized examples
- Expected versus observed behavior
- The date and approximate time you noticed the issue
- Your preferred contact method for follow-up

## Scope notes

- There is no bug bounty program described in this repository.
- No response-time guarantees are made here.
- This document does not claim security certifications or unsupported technical assurances.
