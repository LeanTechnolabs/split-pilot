# Changelog

What’s new in Split Pilot.

## 2026-06-27 - v1.1.0

### Added

- **Record Payment Prefill from Recommended Transfers** - the balances panel shows a compact **Record payment →** link under each recommended transfer and routes to the Add Transfer screen to quickly record a payment for suggested settlements

## 2026-06-24 - v1.0.1

### Improved

- Added start and end date constraints to the trip form

## 2026-06-22 - v1.0.0

### Added

- Initial public release with core trip expense tracking
- Create a trip and share it through a private trip link
- Add members to a trip
- Record expenses and transfers
- Split expenses equally, by parts, or by fixed amounts per participant
- Participant balance calculations and recommended settlement transfers
